# 数学ライブラリ

このライブラリはLazeで数学的処理をできるようにします。

## クラス

このライブラリにあるクラスの一覧:

### [ベクトル3D](/lib/3d/vec3)

3次元上の座標や色のRGBを示すために使います。

### [行列4x4](/lib/3d/matrix4x4)

4x4の行列です。3Dライブラリの内部に使われています。

### [三角関数](/lib/3d/trig_function)

数学的な三角関数です。3Dライブラリの内部に使われています。

### [絶対値](/lib/3d/abs)

絶対値を返す関数です。

### [ラジアンへ](/lib/3d/toRad)

360度法から弧度法に変換する関数です。

### [平方根](/lib/3d/sqrt)

平方根を返す関数です。

このクラスは3Dのグラフィックスで使われています。
